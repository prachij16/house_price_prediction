# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the **price of a house** based on different features such as:
* House Size
* Number of Bedrooms
* Age of the House
The project uses a **Machine Learning model** to learn patterns from the dataset and predict the price of a new house.
---
## 🚀 Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
---
## ⚙️ How It Works

1. The dataset containing house information is loaded.
2. Features like **Size, Bedrooms, and Age** are used to train the model.
3. The model is trained using **Linear Regression**.
4. The program asks the user to enter new house details.
5. The trained model predicts the **house price**.
---
## 📊 Example Output

Enter house size: 2000
Enter number of bedrooms: 3
Enter house age: 5

Predicted House Price: 380000
